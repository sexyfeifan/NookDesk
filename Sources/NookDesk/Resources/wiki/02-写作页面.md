# 写作页面

## 概述

写作页面是 NookDesk 的核心功能，用于创建和编辑博客文章。

## Astro 模式

当检测到 Astro 项目时，写作页面自动切换为 Markdown 编辑模式：

### 左侧：文章列表
- 显示 `src/content/blog/` 目录下的所有 .md 文件
- 支持搜索和筛选
- 点击文章进入编辑

### 中间：Markdown 编辑器
- 支持中文输入法
- 右键菜单支持 Markdown 格式化
- 实时保存

### 右侧：文章属性
- 标题、发布日期、描述
- 分类（单选）、标签（多选）
- 封面 Emoji、卡片颜色、阅读时间
- 草稿开关

## 新建文章

1. 点击「+ 新文章」
2. 填写标题和文件名
3. 选择 frontmatter 格式（Astro 默认为 Astro YAML）
4. 点击「创建」
5. 在编辑器中撰写内容
6. 点击「保存并发布」

## 操作按钮

- **保存** — 仅保存到本地文件
- **保存并发布** — 保存并推送到 GitHub
- **删除** — 删除文章文件

## AI 写作助手

点击工具栏的 AI 按钮，可以：
- 生成文章大纲
- 润色段落
- 生成摘要
- 翻译内容
