# 页面编辑

## 概述

页面标签页用于编辑博客的非文章内容：站点配置、项目展示、友情链接。

## Astro 页面编辑器

当检测到 Astro 项目时，页面编辑器显示三个区块：

### 站点配置
- 站点标题（修改 `src/layouts/Base.astro` 中的 `<title>`）
- 站点描述

### 项目展示
- 编辑 `src/pages/projects.astro` 中的 `const projects` 数组
- 每个项目包含：emoji、名称、描述、URL、技术栈
- 支持添加/删除项目

### 友情链接
- 编辑 `src/pages/friends.astro` 中的 `const friends` 数组
- 每个友链包含：头像（emoji 或 URL）、名称、描述、URL
- 支持添加/删除友链

## Vite 页面编辑器

如果检测到 Vite + React 项目，可以编辑：
- 品牌信息
- 英雄区（打字机动画、头像）
- 技能标签
- 统计数据
- 关于我
- FAQ

## 保存

每个区块独立保存，修改后点击对应的「保存」按钮。
