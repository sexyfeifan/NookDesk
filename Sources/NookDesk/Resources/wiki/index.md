## 支持的博客框架

- **Astro**（推荐）— 使用 Markdown + YAML frontmatter，动森风格主题
- Hugo — 使用 Markdown + TOML/YAML frontmatter
- Vite + React — 使用 TypeScript (posts.ts)

## 功能概览

- Markdown 文章管理（分类、标签、封面、颜色）
- 页面编辑（站点配置、项目展示、友链）
- 一键发布到 GitHub Pages
- AI 写作助手
- 动森风格 UI
