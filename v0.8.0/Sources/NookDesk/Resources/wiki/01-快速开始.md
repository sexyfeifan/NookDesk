# 快速开始

## 什么是 NookDesk？

NookDesk 是一个 macOS 桌面应用，帮助你管理 Astro 博客项目，撰写 Markdown 文章，并一键发布到 GitHub Pages。

## 首次启动

1. 启动 NookDesk，进入引导流程
2. 选择「创建新博客」或「选择已有目录」
3. 如果创建新博客，内置 Astro 主题会自动注入到指定目录
4. 配置 GitHub 仓库地址和 Token
5. 完成设置，开始写作

## 创建新博客

点击「创建新博客」，选择保存路径，NookDesk 会：
- 注入内置 Astro 动森风格主题
- 初始化 Git 仓库
- 安装 npm 依赖
- 自动检测项目类型

## 从 GitHub 克隆

如果你已有博客仓库：
1. 点击「从 GitHub 克隆」
2. 填写仓库 URL 和本地路径
3. 等待克隆完成

## 文章格式

Astro 博客使用 Markdown + YAML frontmatter：

```markdown
---
title: "文章标题"
description: "文章描述"
pubDate: 2026-06-16
category: "技术"
tags: ["Astro", "博客"]
cover: "🏝️"
color: "app-blue"
readTime: "5 分钟"
draft: false
---

正文内容...
```

## 发布

1. 写完文章后点击「保存并发布」
2. 切换到「发布」标签页
3. 点击「一键发布」
4. 等待 GitHub Actions 部署完成
