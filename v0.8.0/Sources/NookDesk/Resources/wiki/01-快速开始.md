# 01-快速开始

## 软件定位

NookDesk 是一个面向 Vite + React 博客项目的桌面工作台，核心目标是把写作、配置、资源管理和发布集中到一个界面中完成。

## 首次使用建议顺序

1. 在「项目设置」中选择博客项目根目录
2. 确认构建工具、远程仓库地址和 Token
3. 在「主题设置」中检查站点配置
4. 进入「写作」页，新建并保存一篇测试文章
5. 进入「发布」页做一次完整预检

## 内容形态说明

### TypeScript 文章（posts.ts）
Vite + React 项目使用 `src/pages/Home/posts.ts` 存储文章数据，支持结构化编辑。

### Markdown 正文
每篇文章的正文支持 Markdown 格式，可使用 Vditor 编辑器或兼容模式。
