# 01-快速开始

## 软件定位

HugoDesk 是一个面向 Hugo 项目的桌面工作台，核心目标是把写作、配置、资源管理和发布集中到一个界面中完成。

## 首次使用建议顺序

1. 在“项目设置”中选择 Hugo 项目根目录
2. 确认 Hugo 可执行文件、远程仓库地址和 Token
3. 在“主题设置”中检查主题识别是否正确
4. 进入“写作”页，新建并保存一篇测试文章
5. 进入“发布”页做一次完整预检

## 内容形态说明

### 单文件
适合普通博客文章。

### 页面包（Leaf Bundle）
适合文章与图片、PDF、附件放在同一目录中管理。

### 栏目包（Section Bundle）
适合作为栏目首页、文档目录和聚合入口。
